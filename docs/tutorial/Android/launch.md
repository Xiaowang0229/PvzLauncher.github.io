# 启动游戏

## 启动前准备

本章节将指导您完成**植物大战僵尸**游戏的启动流程。注意，手机版不支持单独启动修改器

### 启动界面概览

当游戏库中尚未导入任何游戏时，启动界面如下所示。此时「启动游戏」按钮处于禁用状态。

![未安装游戏与修改器时的启动页面](/android/launch-page-01.png)

只有在完成以下任一操作后，相应启动按钮才会变为可用状态：

* 通过[管理](/tutorial/Android/manage.md)页面成功导入游戏版本  
* 通过[下载](/tutorial/Android/download.md)页面完成游戏资源的下载与安装  
* 将目标游戏版本设置为**活动状态**

![已安装游戏与修改器时的启动页面](/android/launch-page-02.png)

## 启动操作

启动相关控件位于主界面右下角，共包含两个主要按钮：

### 1. 启动游戏

* 按钮下方会显示当前处于**活动状态**的游戏版本名称  
* 点击「启动游戏」按钮，即可启动所选版本的《植物大战僵尸》
