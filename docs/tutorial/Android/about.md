# 关于 PvzLauncher

## 概述

**Plants vs. Zombies Launcher for Android（植物大战僵尸启动器安卓版）** 是一款专为安卓版《植物大战僵尸》系列玩家设计的版本管理与启动工具，提供以下核心功能：

- 高速下载与更新支持  
- 便携式多版本管理  
- 一键快速启动与切换  

旨在显著提升玩家在多版本共存环境下的使用效率与便捷性。

## 适用场景

若您仅长期游玩单一版本的《植物大战僵尸》，本启动器的功能可能无法完全体现其价值。

**PvzLauncher 主要服务于以下用户群体：**

需要频繁体验多个官方版本、社区汉化版或各类第三方改版（MOD）的玩家。通过本工具，您可以快速下载、安装、整理并在不同版本之间自由切换，从而大幅提高多版本并存场景下的管理效率与使用体验。

## 历史沿革

本文档所描述的产品为采用 Kotlin 技术后期重构的 **Plants vs. Zombies Launcher for Android**（植物大战僵尸启动器）。

其初期是基于 Microsoft MAUI 框架开发的初代 **Plants vs. Zombies Launcher**，早期界面已无法找到。

随着开发组夜以继日的开发和对安卓生态环境的深入了解，发现 MAUI 技术栈已难以满足当前要求。因此，项目组决定采用 Kotlin 技术栈对启动器进行全面重构。力求提供更流畅、更符合当代审美与使用习惯的体验。
