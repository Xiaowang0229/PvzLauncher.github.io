# 游戏管理

本章节指导您完成《植物大战僵尸》游戏及修改器的导入、设置与日常管理操作。

首次使用启动器时，游戏库为空，管理界面显示如下：

![空游戏库的管理页面](/android/manage-page-01.png)

## 添加游戏或修改器

支持两种添加方式：在线下载或本地导入。

### 通过下载中心添加

您可在下载页面浏览并获取官方支持的游戏版本及常用修改器。详细步骤请参阅[下载教程](/tutorial/Android/download.md)。

### 通过本地导入添加

1. 在管理页面右上角点击「导入」按钮，随即进入导入页面

   ![导入页面](/android/manage-page-02.png)

2. 选择已经安装的应用程序作为游戏，点击右侧的箭头，输入游戏的显示名称

   ![导入对话框 - 输入名字](/android/manage-page-03.png)

3. 在弹出的对话框点击确认后，该项消失。游戏导入成功

   ![游戏导入成功](/android/manage-page-04.png)

## 游戏与修改器管理

### 设置活动状态

- 点击右侧箭头进入管理页面，单击设为活动按钮，即可将其设置为**活动状态**。
![设为活动状态](/android/manage-page-05.png)  
- 设置成功后，卡片右下方将显示「活动」标识。

  ![已设为活动状态的游戏卡片](/android/manage-page-06.png)

- 只有处于活动状态的游戏/修改器才能在[启动页面](/tutorial/Android/launch.md)被正常启动。

### 管理游戏

右键点击任意游戏卡片，将打开详细设置页面。您可在此查看信息或执行以下操作：

![游戏详细设置界面](/android/manage-page-05.png)

- **设为活动**  
  将该游戏设为主界面启动的活动游戏。

- **删除游戏**  
  执行删除操作需输入名称确认，以防止误操作。  
  ![删除确认对话框](/android/manage-page-07.png)

- **更改名称**  
  修改游戏在启动器中显示的名称。
