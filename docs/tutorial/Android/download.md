# 下载游戏

本章节指导您完成《植物大战僵尸》游戏及修改器的下载操作。

我们拥有一个丰富且持续更新的游戏库。截至本文编写时，游戏库已收录 **40+** 款可下载内容（包括**原版、改版和修改器**）。

所有资源均可通过启动器**免费、高速**下载，并在下载完成后自动安卓，无需手动操作。

如果你没有在游戏库中找到想要的改版，欢迎通过 [Issue](https://github.com/PvzLauncher/PvzLauncher.Android/issues/new) 提交申请，我们会尽力将其添加到游戏库中。

![download-detail](/android/download-page-01.png)

首先，确定好你要安装的游戏版本，并点击这个版本以进入详细信息页面

![download-info](/android/download-page-02.png)

点击右侧绿色的下载按钮，就可以开始准备下载了

![download-task](/android/download-page-03.png)

有些游戏可能不止一个版本，例如展示的游戏版本分带修改器无限阳光版本和不带修改器的原版，您可以依据自身情况进行选择

![download-task](/android/download-page-04.png)

接下来，输入这个游戏的名字，点击确定后，游戏就会被加入任务列表。

![download-task](/android/task-page.png)

等待全自动下载安装完毕后，此版本就会作为可以启动的游戏版本被加入管理列表内，您就可以选中它并启动了，选中操作详见[游戏管理](/tutorial/Android/manage.md)
